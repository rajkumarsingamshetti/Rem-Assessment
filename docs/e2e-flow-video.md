## E2E Flow Demo (Loom)

This video demonstrates the complete end-to-end user journey using Playwright automation, including both **positive and negative test scenarios**.

### Positive Scenarios

* User successfully navigates to the website
* Selects skip hire options
* Completes the booking flow successfully

### Negative Scenarios

* Validation errors for missing or incorrect inputs
* Handling invalid selections or edge cases
* Error messages and UI behavior verification
